# Builder
- When construction gets a little bit too complicated.

## Motivation
- Some objects are simple and can be created in a single constructor all
- Other objects require a log of ceremony to create.
- Having an object with 10 constructor argumnets is not productive.
- Instead, opt for piecewise construction.
- Builder provides an APIs for constructing an object step-by-step.

-> When piecewise object construction is complicated provide an API for doing it succinctly.
